Politique de confidentialité

Cette politique explique comment les données sont collectées et utilisées lorsque vous nous contactez via le formulaire.

1. Données collectées

Nous collectons les informations que vous renseignez dans le formulaire de contact : nom, email, téléphone (facultatif), profil, type de projet et message.

2. Finalité

Ces informations sont utilisées uniquement pour répondre à votre demande, organiser un échange et préparer une proposition.

3. Conservation

Les messages sont conservés le temps nécessaire au suivi de votre demande, puis supprimés sur simple demande.

4. Partage

Les données ne sont pas revendues. Elles peuvent être traitées par nos fournisseurs techniques (ex: hébergement, email) uniquement pour permettre le service.

5. Vos droits

Vous pouvez demander l’accès, la modification ou la suppression de vos données en écrivant à l’adresse indiquée dans la page Contact.
